📖 名詞解釋（股票新手必讀）

MA（均線）
過去 N 天收盤價平均值，用來判斷趨勢方向。MA5 是 5 日線（短線）、MA20 是月線（中線）、MA60 是季線（長線）。價格在均線上方代表趨勢偏多。

多頭排列
短期均線在長期均線上方（MA5 > MA10 > MA20 > MA60），代表上漲趨勢確立，適合做多。

RSI（相對強弱指數）
衡量漲跌動能的指標，範圍 0-100。> 70 表示「超買」（漲太多可能回檔），< 30 表示「超賣」（跌太多可能反彈）。

MACD
趨勢動能指標。「金叉」（MACD 線上穿信號線）為買入信號，「死叉」（下穿）為賣出信號。柱狀體由負轉正代表動能回升。

布林通道
價格在上下軌之間波動。觸及上軌可能過熱，觸及下軌可能超跌。通道收窄代表即將變盤。

支撐位
價格下方的「地板」，跌到這裡容易止跌反彈。跌破支撐可能加速下跌。

壓力位
價格上方的「天花板」，漲到這裡容易遇阻回落。突破壓力可能加速上漲。

PCR（Put/Call Ratio）
期權市場看跌/看漲的比例。> 1 代表市場偏避險（偏空），< 0.7 偏樂觀（偏多）。

空單比例
被借來賣出（看跌）的股票佔流通股比例。偏高代表市場看空情緒重，但也可能觸發「軋空」（空頭回補推升股價）。

Beta
股票相對大盤的波動度。Beta > 1 代表波動比大盤大（高風險高報酬），< 1 較穩定。

P/E（本益比）
股價 ÷ 每股盈餘。越高代表市場越看好未來成長，但也越「貴」。科技股通常 P/E 較高。

停損
預設的最大虧損價位，跌到就賣出，避免更大損失。紀律停損是投資最重要的規則。

停利
預設的獲利目標價位，到了就賣出鎖定利潤。「會賣才是師傅」。

量能
成交量的大小。上漲配合放量代表真突破，上漲但縮量可能是假突破。下跌縮量代表賣壓減輕。

52W 位置
目前價格在過去一年最高最低之間的相對位置。接近 100% 代表在高點附近，接近 0% 代表在低點。

分析師目標價
華爾街分析師預測的未來股價。「目標價空間」是平均目標價與現價的差距百分比，正數代表分析師認為還有上漲空間。

板塊相對強度
個股漲跌幅 vs 所屬板塊 ETF 漲跌幅的比較。「超額報酬」為正代表跑贏板塊（強勢股），為負代表落後板塊（弱勢股）。

財報日
公司公布季度營收的日期。財報前後股價波動通常較大，業績超預期會大漲，不如預期會大跌。新手建議避開財報日重倉。

短線信號燈
綜合 RSI、MACD柱方向、量價背離、布林帶位置等指標的計分系統。🟢(≥30分)=買進機會、🔴(≤-30分)=該停利了、🟡=留意中、⚪=觀望。分數越極端信號越強。

量價背離
股價創新高但成交量反而縮小，代表追買的人變少了，上漲可能撐不久。這是最重要的頂部預警信號之一。

MACD 柱狀體方向
柱子從大變小 = 趨勢動能在衰退（不管漲跌），這個比等金叉/死叉更早發出預警。柱子由負轉正 = 空頭動能耗盡、多頭開始。

⚡ 短線信號系統說明（1-5 天波段操作）

🟠 短線信號是什麼？
綜合 7 層技術指標計算的即時計分系統（-100 ~ +100），告訴你現在這個瞬間是買入的好時機還是該離場。它不看趨勢方向，只看「時機」。

🟠 9 層架構怎麼算分？
Layer 0：基礎信號 — MACD 柱方向（放大/縮小）、金叉死叉、量價背離、布林帶位置、均線排列、連漲連跌天數
Layer 1：動量指標 — RSI 和 KD 取較極端的那個。RSI > 75 = 過熱扣分，RSI < 25 = 超賣加分
Layer 2：確認層 — OBV 背離確認 + K 線形態（錘子、吞噬、晨星等）。只有 Layer 1 有方向時才加分
Layer 3：價位層 — VWAP 支撐/壓力 + 費氏回撤位
Layer 4：MFI 過濾 — 資金流量極端值（> 80 過熱，< 20 超賣）
Layer 5：基本面 — 營收成長、獲利成長、分析師評等、目標價（±20 分上限）
Layer 6：題材事件 — 財報日倒數、板塊輪動、軋空題材（±15 分上限）
Layer 7：ConnorsRSI （2026 新增） — 專為短線均值回歸設計，65-75% 勝率。RSI(3) + 連漲跌天數 RSI + 單日漲跌百分位。CRSI < 10 = 極度超賣（+12 分），CRSI > 90 = 極度過熱（-10 分）。只在 SMA200 上方做多
Layer 8：RVOL 量能確認 （2026 新增） — 1800+ 突破事件回測：RVOL 1.5-2.0x = 最佳確認（+10 分），放量下跌 = 風險（-8 分），上漲無量 = 不可靠（-3 分）

🟠 ADX 乘數是什麼？
技術面分數會乘上 ADX 係數。ADX < 15（完全盤整）→ 乘 0.4，信號打四折；ADX > 40（強趨勢）→ 乘 1.2，信號加強。意思是：沒有趨勢時，所有技術信號都不可信。

🟠 信號等級怎麼看？
📈 強勢做多（≥70） — 多指標共振，技術面強勢，可分批佈局
📈 偏多（≥50） — 信號偏多，可考慮分批建倉
📊 低接留意（30-49） — 出現底部回升訊號，可開始關注（青色 ≠ 風險）
➡️ 中性觀望（-14~14） — 訊號不明確，等待更好機會
📉 偏弱（-15~-29） — 頂部訊號浮現，可先減碼（橙色 = 警告）
📉 弱勢離場（≤-30） — 短線風險高，建議降低曝險

🟠 K 線形態有哪些？
🔨 錘子線 — 長下影線，止跌信號。在跌勢末端出現 +25 分，漲勢中只 +5 分（位置很重要！）
🌠 射擊之星 — 長上影線，反壓信號。在漲勢末端 -25 分
🐂 多頭吞噬 — 前黑後紅且完全包住，底部反轉 +18 分
🐻 空頭吞噬 — 前紅後黑且完全包住，頭部反轉 -18 分
⭐ 晨星 — 大陰 + 十字 + 大陽，底部強反轉 +22 分
🌙 夜星 — 大陽 + 十字 + 大陰，頭部強反轉 -22 分
💥 爆量 — 成交量 > 2 倍均量，配合 K 線方向加減 15-20 分

🟠 指標衝突檢測
當 RSI 和 MACD 方向矛盾、或布林帶位置和 RSI 分歧時，系統會自動偵測並扣分。例如：RSI 偏高但 MACD 柱為負 = 動能不一致，扣 5 分。指標互相打架 = 信號不可靠。

🟠 信號穩定度（信心指標）
看過去 5 天的 RSI 和 MACD 方向是否一致：
🟢 信號穩定（≥75%）— 多指標方向一致，信號可靠
🟡 信號普通（50-75%）— 部分指標分歧
🔴 信號不穩（<50%）— 指標方向矛盾，可能是噪音

🟠 VIX 市場體制
大盤恐慌指數會影響所有信號的可信度：
🟢 正常（VIX 15-25） — 標準權重
🟠 高波動（VIX 25-35） — 技術信號打七折
🔴 恐慌（VIX > 35） — 技術信號打五折，風險權重 ×1.5
🔵 低波動（VIX < 15） — 過度樂觀，注意風險

🏦 籌碼面分析系統說明（大戶在幹嘛？）

🟣 籌碼分析是什麼？
看「聰明錢」（法人、機構、大戶、內部人）的動向。技術面告訴你價格怎麼走，籌碼面告訴你誰在買、誰在賣。大戶站多方 = 有靠山，回檔有人接；大戶出貨 = 小心崩盤。

🟣 籌碼分數怎麼算？（-100 ~ +100，7+2 維度）

① 機構持股比例（±20）— 來源：Gompers & Metrick (2001)
• > 80% = +20 — 大型基金重倉，股價有「護城河」
• 60-80% = +12 — 法人認可
• 40-60% = +5 — 中等水準
• < 40% = -10 — 法人不愛，流動性可能差

② 空單 + DTC + 軋空偵測（±25）— 來源：Desai et al. (2002)
• < 3% = +15 — 幾乎沒人看空
• 5-10% = -5 — 看空情緒升溫
• 10-20% = -15 — 高空單（每月異常報酬 -0.76~-1.13%）
• DTC > 8 天 — 空頭極度擁擠。如果同時 RSI 超賣 + MACD 翻正 → 🚀 軋空潛力，分數反轉加分！

③ PCR 含極端反轉（±15）— 升級：多到期日 + OI 混合
計算方式：前 3 個到期日加總，40% 交易量 + 60% 未平倉量（OI 更穩定）
• 0.5-0.8 = +15 — 適度樂觀，最健康
• < 0.4 = -3 — 極度樂觀 = 反向過熱警告
• 1.2-1.5 = -15 — 大量買 Put 避險
• > 1.5 = +5 — 極度恐慌 = 反向底部信號（所有人都在恐慌的時候，往往是底部）

④ Beta 波動係數（±5）
• < 1.0 = +5（穩健）｜> 2.0 = -5（暴漲暴跌，高風險）

⑤ 內部人交易（±15）— 來源：Lakonishok & Lee (2001) → 2025 更新
• 大額密集買入 ≥2 筆 (>$100K) = +15 — 2025 升級：過濾 option exercise + 金額加權
• 密集買入 ≥3 筆 = +12 — Cluster buying 仍有效
• 密集賣出 ≥5 筆 = -10 — 2023 Rule 10b5-1 修訂後賣出信號已減弱

🟣 籌碼燈號怎麼看？
🚀 軋空潛力 — 高空單 + 超賣 + MACD 翻正，空頭回補可能推升股價爆發
🟢 籌碼強勢（≥40） — 法人重倉、空單低、內部人買入，最佳籌碼結構
🟢 籌碼偏多（20-39） — 整體偏多，回檔有人接盤
➡️ 籌碼中性（0-19） — 沒有明顯偏向
📉 籌碼偏弱（-1~-20） — 部分警訊，注意內部人動向
🔴 籌碼偏空（< -20） — 法人持股低 + 空單高 + 內部人賣出，倉位不宜過重

🟣 內部人交易的學術演進
經典研究（仍有效）： Lakonishok & Lee (2001) 發現內部人買入月均超額 +0.5%；Kang et al. (2018) 發現 Cluster buying 效果翻倍達 +2%/月。
2025 最新更新：
• Alpha Decay 研究（2025）— 70-80% 的超額報酬發生在「交易日→公開申報日」之間（此時資訊還是非公開的），等散戶看到 Form 4 時 alpha 已大幅衰減
• Virginia Tech 研究（JFQA 2025）— 內部人會利用「散戶注意力」擇時：散戶搜索量高時賣出、低時買入，利用情緒定價錯誤
• Rule 10b5-1 修訂（2023 後）— 預設計畫交易的資訊含量下降，只有非計畫性買入才是真信號
• 結論：Cluster buying（多位 C-Suite 同期買入）仍然有效，但單筆內部人交易的 alpha 已經比 20 年前小很多
我們的做法：只看「密集買入」（≥2 人），不依賴單筆交易。這是 2025 研究認可的最穩健策略。

🟣 什麼是軋空（Short Squeeze）？
當空頭太多（空單比例 > 10%）+ 股價突然上漲 → 空頭被迫回補 → 推升股價 → 更多空頭回補 → 連鎖反應。
經典案例：2021 年 GameStop（GME）$20 → $483。

2024-2025 學術共識：
• DTC 和空單比例單獨不能預測軋空 — 它們只是「燃料」，還需要「火花」（催化劑）
• 軋空三要素：① 高空單/高 DTC（燃料）+ ② 借券困難/費用高（約束）+ ③ 正面催化劑/散戶注意力飆升（火花）
• 高空單本身是看空信號（空頭通常是專業投資人，他們的判斷常常是對的）
• 只有當三要素同時出現，高空單才從「利空」翻轉為「軋空機會」
系統偵測 RSI 超賣 + MACD 翻正作為催化劑代理，配合高空單/高 DTC 觸發 🚀 標籤。

🟣 籌碼面和技術面怎麼搭配？
✅ 最佳 = 技術面多頭 + 籌碼偏多 + 內部人買入 → 三重確認，安全加碼
⚠️ 注意 = 技術面多頭 + 內部人密集賣出 → 可能是頂部信號，大戶在倒貨
🤔 觀察 = 技術面偏空 + 籌碼偏多 → 大戶還沒走，可能只是洗盤
🚀 特殊 = 技術面超賣 + 高空單 + 動能翻正 → 軋空機會，可小倉位博反彈
🚨 危險 = 技術面空頭 + 籌碼偏空 + 內部人賣出 → 三重利空，趕快離場

📈 中期趨勢系統說明（Stage / Minervini / RS）

🔵 Weinstein Stage（階段分析）是什麼？
把股票的一生分成「春夏秋冬」四個階段，由 150 日均線（SMA150）的方向決定：
Stage 1 底部打底🔄 — SMA150 走平或下降，股價在附近來回。像冬天，什麼都不做，等春天。
Stage 2 上升趨勢📈 — SMA150 上升，股價在上方。像夏天，唯一應該買入的階段。Minervini、CAN SLIM 都只在這階段操作。
Stage 3 做頭📉 — SMA150 走平，股價開始橫盤震盪。像秋天，該開始準備離場。
Stage 4 下跌趨勢⬇️ — SMA150 下降，股價在下方。像冬天，絕對不要買，等跌完再說。

🔵 Minervini SEPA 8 項篩選
美股冠軍交易員 Mark Minervini 的趨勢模板，8 項全過 = 頂尖趨勢股。條件包括：股價在 SMA50/150/200 上方、均線正確排列（SMA50 > SMA150 > SMA200）、SMA200 上升中、距 52 週高點 25% 以內、距 52 週低點 30% 以上。能通過 8/8 的股票只佔全市場約 5%。

🔵 RS 相對強度 （升級：Mansfield RS）
不再是簡單的「3個月超額報酬」，改用 Mansfield RS：將股價/大盤比值正規化到自己的 252 日均線。
MRS > 0 = 正在跑贏大盤 | MRS 上升 = 加速跑贏（最佳狀態）
優勢：跨股票可比較、消除量級差異、偵測到加速vs減速的微妙變化。

🔵 趨勢品質 （升級：Q + Efficiency Ratio）
原有 Q-Indicator（淨位移/總路徑）加入 Kaufman Efficiency Ratio（60 日）取平均。
ER ≈ 1 = 完美直線趨勢 | ER ≈ 0 = 極度震盪 | 兩者結合 = 更穩定的趨勢品質評估。

🔵 Stage 2 量能確認 （2026 新增）
Stage 2 突破如果沒有量能支撐，容易是假突破。系統偵測 RVOL（最近 5 日 vs 20 日均量）：
RVOL > 1.5x = 量能確認，突破有效 | RVOL < 0.7x = 量能偏弱，扣 3 分

🔵 中線分數怎麼看？
六個維度加總 -100 ~ +100：
① MA 排列（±30）：均線是否正確排列 + 斜率是否向上
② Mansfield RS（±20）：正規化相對強度
③ ADX 趨勢（±15）：趨勢強不強 + 方向
④ 量能確認（±15）：成交量有沒有跟上趨勢 + Stage 2 量能檢查
⑤ 動量 ROC（±10）：1個月+3個月漲跌幅
⑥ 趨勢品質（±10）：Q-Indicator + Efficiency Ratio 混合

🔵 短線 vs 中線怎麼搭配看？
⭐ 最佳買點 = 中線強勢（≥60）+ 短線買入（≥30）→ 趨勢 + 時機都到位，積極分批買進！
✅ 回檔加碼 = 中線強勢（≥60）+ 短線過熱/觀望 → 趨勢沒壞，短線拉回就是撿便宜的機會
🟡 小心操作 = 中線觀望 + 短線買入 → 只能小量試單，設嚴格停損
🚨 不要碰 = 中線空頭（< -10）→ 不管短線看起來多便宜，趨勢向下就是不碰

🎯 一句話總結：怎麼用這套系統買股？

1. 先看中線分數 — 只買中線 ≥ 30（偏多或強勢）的股票，中線 < 10 的不碰
2. 再看 Stage — 只在 Stage 2（上升趨勢）買入，Stage 4（下跌）絕不抄底
3. 確認 Minervini — 通過 ≥ 6/8 項才是好趨勢股，8/8 是精品
4. 等短線時機 — 中線確認好後，等短線信號 🟢（≥30 分）出現再進場
5. 設好停損 — 跌破 SMA50 減碼，跌破 SMA200 出場
記住：中線決定「買不買」，短線決定「什麼時候買」
